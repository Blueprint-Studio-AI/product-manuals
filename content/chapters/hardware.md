## Power up

1. Hold the **power** button for 2-3 seconds until the Green LED light at the top of the scanner illuminates. 

2. Wait 45-60 seconds for the scanner to boot.

{% callout type="note" %}
Do not switch off in normal use. Just plug the unit into a wall outlet at the end of the night. Only switch off when storing the unit.
{% /callout %}

## Charging

{% split image="/images/idvisor3-charging.png" imageAlt="IDVisor 3 charging in cradle" side="right" %}

- Only use the supplied 5V USB power&nbsp;supply.
- It takes about 4 hours to fully charge.
- When device is charging, the LED indicates the charge status. Green = full. Red = charging.

{% /split %}

{% callout type="note" %}
Leave unit charging when not in use. Batteries will not be overcharged and unit will be ready for portable use at all times.
{% /callout %}

## Find my device

{% split image="/images/idvisor3-find-device.png" imageAlt="Find My Device map view with IDVisor 3" side="right" %}

Google's 'Find My Device' feature is a great way to find, lock, or erase all data if you misplace the scanner and want to keep its contents from the wrong hands.

To install Google's Find My Device on your IDVisor 3, view the setup guide by [clicking here](https://support.tokenworks.com/portal/en/kb/articles/how-to-install-google).

{% /split %}

{% callout type="important" %}
Save the Google account and PIN used on your ID Scanner. Losing this information may permanently lock the device. If an IT firm sets it up, ensure they share these details with you. Locked devices must be returned to TokenWorks for a factory reset. A fee applies, and all previously scanned data will be permanently erased.
{% /callout %}

## Change PIN Code

The unit comes with a pre-initialized PIN code (5678) for the following functions: Database, Configuration, Tag List, and Close Application. You can change the PIN from the default by navigating to **Setup > Configuration > Advanced > Change Pincode**.

Enter the current PIN code, then the new PIN code, and confirm by re-entering the new PIN code. Press Save when done.
