# IDVisor 3 User Manual

{% /cover %}

## Introduction

This document describes how to use the TokenWorks IDVisor 3 ID Scanner. Building upon TokenWorks' classic military-grade durability and color-coded alerts, the IDVisor 3 is TokenWorks' fully featured Android-based age verification ID Scanner, adding a high-performance point-and-shoot style barcode scanner, an AI integrated high resolution camera, and an Immersive Touch Surface, along with audible alerts, customer tagging, and more.

The IDVisor 3 supports all US and Canadian driver licenses, state issued ID cards, plus government and military issued IDs. It provides quick age verification by simply pointing the scanner at an ID's 2D barcode. The device quickly calculates age based on the date of birth read from the ID and clearly indicates to the operator if the person is of age. Multiple age categories are supported for businesses selling products or services with different age limits, such as 18/21 for entry/alcohol, etc.
