## Introduction

This document describes how to use the TokenWorks IDVisor 3 ID Scanner. Building upon TokenWorks' classic military-grade durability and color-coded alerts, the IDVisor 3 is TokenWorks' fully featured Android-based age verification ID Scanner, adding a high-performance point-and-shoot style barcode scanner, an AI integrated high resolution camera, and an Immersive Touch Surface, along with audible alerts, customer tagging, and more.

The IDVisor 3 supports all US and Canadian driver licenses, state issued ID cards, plus government and military issued IDs, including CAC (Common Access Card). It provides quick age verification by simply pointing the scanner at an ID's 2D barcode. The device quickly calculates age based on the date of birth read from the ID and clearly indicates to the operator if the person is of age. Multiple age categories are supported for businesses selling products or services with different age limits, such as 18/21 for entry/alcohol, etc.

Optional Add-ons/Upgrades include:

- VIP/Banned Upgrade
- Anti-PassBack Upgrade
- Photos Upgrade
- Passport Upgrade

### b. Power up

Hold the power button for 3 seconds until the screen lights up. Wait 45-60s for the device to start up.

### c. Charging

{% split image="/images/idvisor3-charging.png" imageAlt="IDVisor 3 charging in cradle" side="right" %}

1. Only use the supplied 5V USB power supply.
2. It takes about 4 hours to fully charge.
3. When device is charging, the LED indicates the charge status. Green = full. Red = charging.

{% /split %}

{% callout type="note" %}
Leave unit charging when not in use. Batteries will not be overcharged and unit will be ready for portable use at all times.
{% /callout %}

### d. Find my device

{% split image="/images/idvisor3-find-device.png" imageAlt="Find My Device map view with IDVisor 3" side="right" %}

Google's 'Find My Device' feature is a great way to find, lock, or erase all data if you misplace the scanner and want to keep its contents from the wrong hands.

[Install Google's Find My Device on your IDVisor 3](https://www.google.com/android/find)

{% /split %}

{% callout type="important" %}
Save the Google account and PIN used on your ID Scanner. Losing this information may permanently lock the device. If an IT firm sets it up, ensure they share these details with you. Locked devices must be returned to TokenWorks for a factory reset. A fee applies, and all previously scanned data will be permanently erased.
{% /callout %}
